Preprint available on the arXiv here: [https://arxiv.org/abs/1702.05652](https://arxiv.org/abs/1702.05652)

## Installation

    $ pip install -e .

    $ python setup.py build_ext --inplace

Then, you will need to add the `scripts` directory to your `PATH`.
